# Halal Open Beauty Facts

## Project Description
Halal Open Beauty Facts is an open-source project that aims to create a database of halal cosmetics and beauty products. By providing transparent information about ingredients and certifications, we hope to make it easier for consumers to make informed choices about the products they use on their skin and hair.

## Features
- Extensive database of halal-certified beauty products.
- User-friendly interface for searching and filtering products.
- Contribution system for users to add new products and information.
- API access for developers to integrate with their applications.

## Installation
### Prerequisites
- Node.js (version x.x.x)
- NPM (version x.x.x)

### Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/halalopenbeautyfacts/halalopenbeautyfacts.git
   cd halalopenbeautyfacts
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```

## Usage
Once your application is running, you can access it via `http://localhost:3000`. 
- Use the search bar to find products by name, category, or ingredients.
- Navigate to the contributions section to add new products or edit existing ones.
