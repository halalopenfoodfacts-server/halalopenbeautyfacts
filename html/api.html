🔌 API Publique Beauty

Accédez aux données cosmétiques halal via nos endpoints proxy. Données issues d'Open Beauty Facts.

GET Recherche de produits cosmétiques

/proxy/search/search?q={query}&page_size={n}&page={p}&index=beauty
ParamètreTypeDescription
qstringTexte à rechercher
indexstringMettre beauty pour Open Beauty Facts
page_sizeintRésultats par page (max 100)
pageintNuméro de page (base 1)

GET Catalogue par catégorie / pays

/proxy/v2/search?categories_tags={cat}&countries_tags={country}&page_size={n}&page={p}
ParamètreDescription
categories_tagsEx: en:face-creams, en:shampoos
countries_tagsEx: en:france
labels_tagsEx: en:halal

GET Produit par code-barres

/proxy/v2/product/{barcode}.json

Retourne toutes les données d'un produit cosmétique : INCI, labels, certifications, pays de vente.

GET Facettes

/proxy/facets/countries.json
/proxy/facets/categories.json
/proxy/facets/labels.json
